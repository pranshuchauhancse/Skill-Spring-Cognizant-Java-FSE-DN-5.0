# Skill Spring - Cognizant Java FSE Digital Nurture 5.0

Welcome to the **Skill Spring - Cognizant Java Full Stack Engineering (Java FSE) Digital Nurture 5.0** repository.

This repository contains my solutions, hands-on exercises, assignments, and practice programs completed as part of the **Cognizant Digital Nurture (DN) 5.0 Java Full Stack Engineering Training Program**.
